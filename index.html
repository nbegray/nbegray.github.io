Portfolio Examples

Included in this portfolio are examples of applications and projects completed in my Full-Stack Development Certificate Program and also independent projects.

WireFrame CSS Scaffolding

This HTML page utilized only manual CSS framework to generate the floating boxes and sections seen on the page itself. HTML5 and CSS3 were used in the building process for this page.

Taco Giphys GIPHY API Application

This web application utilized Javascript, jQuery, Bootstrap CSS framework, to build an interactive web application. Users can add dynamically add a button to the HTML document and upon clicking the button, the web application searches the GIPHY API database for up to 10 matching queries.

Train Schedule Train Schedule Application

This utilized Javascript, jQuery, Bootstrap CSS framework, MomentJS and Google Firebase database to build an interactive application that builds a functional train schedule. Users can add a train to the application and it will post to a table, which will calculate the upcoming train arrival times.

Project1 - Meteorite Vists Project 1 - Visit a Meteorite App

This application was built as a team using Javascript, jQuery, MomentJS, Google Firebase database, utilizing HTML5 and Materialize CSS framework. Using NASA API and Google Maps API our group built an interactive web application that logs and displays user searches for meteorite impacts anywhere in the world.

LIRI NodeJS App LIRI NodeJS Application

LIRI is a command line NodeJS app that takes in user argument parameters and gives the user back the requested data. This application is using three different APIs - Spotify, Bands in Town and OMDB API databases - to retrieve user requests and return the information in a NodeJS terminal application.

Project 2- Farmer's Market Finder App Project 2 - Farmer's Market Finder App

Farmer's Market Finder is a full-stack development group project created using front-end and back-end technologies. Front End Tech: Bulma, Boostrap4, HTML5, CSS3, JavaScript & jQuery. Back End Tech: Node.js, Express.js, Sequelize, Postman, mySQL2, GitHub & hosted by Heroku.
