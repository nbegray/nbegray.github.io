Portfolio Examples

Included in this portfolio are examples of applications and projects completed in my certificate program and also independent projects completed outside of the program.

WireFrame WireFrame CSS Scaffolding

This HTML page utilized only manual CSS framework to generate the floating boxes and sections seen on the page itself. HTML5 and CSS3 were used in the building process for this page.

Taco Giphys GIPHY API Application

This web application utilized Javascript, jQuery, Bootstrap CSS framework, to build an interactive web application. Users can add dynamically add a button to the HTML document and upon clicking the button, the web application searches the GIPHY API database for up to 10 matching queries.

Train Schedule Train Schedule Application

This utilized Javascript, jQuery, Bootstrap CSS framework, MomentJS and Google Firebase database to build an interactive application that builds a functional train schedule. Users can add a train to the application and it will post to a table, which will calculate the upcoming train arrival times.
